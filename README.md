## Installation

```bash
$ yarn install
```

## Running the app

```bash
$ Add the environment variables in .env file shared
```

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

```bash
$ yarn run test:cov
```
